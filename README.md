## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sinatra-errorcodes'
```

Or install it yourself as:
```bash
$ gem install sinatra-errorcodes
```

## Example
Install gems
```bash
$ bundle install
```

and run it
```bash
$ cd example
$ bundle exec rackup -p 3000
```

## Test
```bash
$ bundle exec rake spec
```
