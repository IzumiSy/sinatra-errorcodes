# Sinatra::Errorcodes
Gem of HTTP error status code class pack

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sinatra-errorcodes'
```

Or install it yourself as:
```bash
$ gem install sinatra-errorcodes
```
